Файлы к статье [Минимальная Arduino-совместимая конструкция за $3](http://mk90.blogspot.com/2011/01/arduino-3-atmega8a-bootloader.html) в блоге [mk90.blogspot.com](http://mk90.blogspot.com)


- `/Arduino-0023` -- файлы для Arduino IDE версии 0023
- `/Arduino-1.0.x` -- файлы для Arduino IDE версии 1.0.x

Для установки скопируйте файлы из соответствующей папки скачанного архива в папку `Sketchbook`. 
Путь можно посмотреть в пункте меню `File -> Preferences Arduino IDE` (`Файл -> Расположение папки со скетчами`).
